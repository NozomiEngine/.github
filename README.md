# Nozomi

**Nozomi** is a modern game engine initiative focused on creator productivity.

We are building Nozomi toward a complete engine platform for:

- 2D
- lightweight 3D
- Web
- Mobile
- Desktop

Our long-term goal is clear:

> Build Nozomi into a true engine product with its own identity, workflow, tools, and ecosystem.

---

## Vision

Nozomi is not intended to remain just a customized upstream fork.

It is being developed toward an independent engine platform with:

- Runtime
- Editor
- Asset Pipeline
- Scripting Workflow
- Build & Packaging Tools
- Debugging / Profiling
- Package / Plugin Ecosystem

We want Nozomi to become:

- more productive than traditional lightweight frameworks
- lighter and more focused in day-to-day development
- suitable for long-term professional project development
- creator-centered and extensible

---

## Positioning

Nozomi is aimed at becoming a:

**modern engine and editor platform for content creation, rapid iteration, and cross-platform deployment.**

We care deeply about:

- fast iteration
- clean project structure
- strong resource workflows
- hot reload friendly architecture
- clear engine boundaries
- better developer experience

---

## Current Focus

Our current priorities include:

- engine architecture cleanup
- resource pipeline foundation
- virtual file system
- unified sync / async asset loading
- build stabilization
- Nozomi-specific module boundaries

This is the phase where Nozomi starts becoming itself.

---

## Repositories

Core repositories in this organization include:

- **Nozomi-Engine-2D** — engine runtime foundation
- **Nozomi-Editor** — future editor and tooling
- **Nozomi-Docs** — documentation
- **Nozomi-Examples** — samples and starter projects
- **Nozomi-Tools** — supporting development tools
- **Nozomi-Packages** — reusable modules and packages

---

## Philosophy

We believe the best engine is not the one with the longest feature list.

It is the one that helps creators:

- build faster
- iterate safely
- understand their project clearly
- ship with confidence

That is the future Nozomi is being built for.

---

## Status

Nozomi is currently under active architectural evolution.

Some parts of the codebase may still reflect earlier foundations while new Nozomi-specific systems are introduced step by step.

This project is not trying to stay a renamed fork.

It is being shaped into a real engine platform.
